Bencodex codec for .NET
=======================

This library implements [Bencodex] serialization format which extends
[Bencoding].

[Bencodex]: https://github.com/planetarium/bencodex
[Bencoding]: http://www.bittorrent.org/beps/bep_0003.html#bencoding
